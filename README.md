# 🎮 Hangman Game (Python)

## 📌 Description
This is a simple text-based Hangman game developed using Python. The player guesses a hidden word letter by letter with a limited number of attempts.

## 🚀 Features
- Random word selection
- 6 attempts limit
- Tracks guessed letters
- Win/Lose condition display

## 🛠 Technologies Used
- Python
- Random module

## ▶️ How to Run
1. Install Python
2. Run the script:
   python hangman.py

## 📷 Sample Output
Displays word with blanks and updates as user guesses.

## 📚 Concepts Used
- Loops (while)
- Conditional statements (if-else)
- Lists & Strings
